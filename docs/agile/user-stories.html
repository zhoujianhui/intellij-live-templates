用户故事

除了用户故事外，还有其它两种不常见的故事类型：

  • User Story
  • Foundation: 开发用户故事所需的基础工作
  • Spike: 预研工作

模版缩写

  • arus: User Story
  • arf: Foundation
  • ars: Spike

如果你想参考示例的话，在上述缩写后，加上"s"，代表"sample"的意思
